# ViralSnipAI Landing Media Placeholders

Phase 1 uses CSS-built product mockups so the landing page ships without heavy media.

Drop future real assets here:

- `hero-demo.webm` or `hero-demo.mp4` - muted product demo loop.
- `hero-poster.jpg` - poster for the hero demo.
- `clip-wall-*.jpg` or `clip-wall-*.webp` - output clip examples.
- `product-dashboard.webp` - product screenshot for the hero/showcase.
- `creator-avatar-*.webp` - real testimonial avatars.

Keep files optimized before committing. Prefer WebP/AVIF for images and compressed WebM/MP4 for video.
